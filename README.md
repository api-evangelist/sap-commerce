# SAP Commerce Cloud (sap-commerce)

SAP Commerce Cloud (formerly SAP Hybris) is an enterprise e-commerce platform providing OCC (Omni Commerce Connect) REST APIs for products, catalogs, carts, orders, customers, and omnichannel commerce experiences. It enables headless commerce, B2C and B2B storefronts, and integrations across the SAP ecosystem.

## APIs

### SAP Commerce Cloud OCC API

The Omni Commerce Connect (OCC) REST API exposes a broad set of commerce and data services, enabling integration of SAP Commerce functionality anywhere in the application landscape. OCC v2 is the default stateless REST API covering products, catalogs, carts, orders, users, and more.

- **Human URL:** [https://help.sap.com/docs/SAP_COMMERCE/e5d7cec9064f453b84235dc582b886da/8c19ab00866910148f87bf32d4a60d38.html](https://help.sap.com/docs/SAP_COMMERCE/e5d7cec9064f453b84235dc582b886da/8c19ab00866910148f87bf32d4a60d38.html)
- **Base URL:** `https://{host}/occ/v2/{baseSiteId}`

### SAP Commerce Cloud Portal API

The SAP Commerce Cloud Portal API provides programmatic access to manage cloud environments, deployments, builds, and configurations within the SAP Commerce Cloud control plane.

- **Human URL:** [https://help.sap.com/docs/SAP_COMMERCE_CLOUD_PUBLIC_CLOUD/452dcbb0e00f47e88a69cdaeb87a925d/66abfe678b55457fab235ce8039dda71.html](https://help.sap.com/docs/SAP_COMMERCE_CLOUD_PUBLIC_CLOUD/452dcbb0e00f47e88a69cdaeb87a925d/66abfe678b55457fab235ce8039dda71.html)
